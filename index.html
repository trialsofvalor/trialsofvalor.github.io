About the Game

Trials of Valor is a real-time roguelike fighting game set in a tough fantasy world filled with monsters, bosses, and deadly challenges. Inspired by classic RPGs, it blends simple class-based characters with fast, skill-based combat.

Features

Real-time combat

Evade, block and attack in real time. Use powerful abilities to your advantage. Perfect timing can be the difference between life and death.

Procedural content

Every run is different with randomized map content and gear modifiers.

Wield for the occasion

Choose the right time to use your primary and secondary weapon.
